# ArchitectureSoftware_Lab2

## Technology Stack:
- JavaScript(V14)
- Jest(Latest)

## Installation 

- Clone the repository
```bash
git clone https://github.com/Er4ik/ArchitectureSoftware_Lab2.git 
```

- Install Node.js and npm on your pc/laptop

## Usage 

For download all dependencies: 

```bash
npm install
```

For run tests:

```bash
npm run test
```
